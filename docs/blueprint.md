# **App Name**: FGPerfume AI Concierge

## Core Features:

- AI Perfume Concierge: Answer user questions exclusively about FGPerfume and its perfumes using admin-provided data via Gemini. Behaves as a closed-domain assistant, providing elegant, confident, and professional responses with sensory descriptions and a sophisticated vocabulary.
- Perfume Data Management: Enable admins to add, edit, update, and delete FGPerfume brand and perfume data, including fragrance notes, inspiration, pricing, and availability.
- Perfume Visibility Control: Allow admins to control which perfumes are visible to users in the application.
- Brand Story Management: Provide a feature for admins to manage the brand story and company information displayed to users.
- User Query Logging: Log user queries (without personal data) for admin review to improve the AI's knowledge base. User data will be stored in a Firestore database.

## Style Guidelines:

- Background color: Soft beige (#F5F5DC) to create a sense of understated luxury.
- Primary color: Deep black (#000000) for a classic, premium aesthetic.
- Accent color: Gold accents (#FFD700) for highlights and key UI elements, conveying opulence.
- Headline font: 'Playfair', a modern serif for an elegant, fashionable, high-end feel; if longer text is anticipated, use 'PT Sans' (sans-serif) for body text.
- Note: currently only Google Fonts are supported.
- Minimalist layout with generous spacing to create an airy and uncluttered interface, enhancing the sense of luxury.
- Subtle animations such as smooth transitions and soft fades to enhance the user experience without being distracting.
